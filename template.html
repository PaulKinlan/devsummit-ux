Minimum target Recommendations

  • iOS: 44 pt
  • Android: 48 dip
  • Microsoft: 23 DLG

What do they mean in web terms?

  • iOS: 1 pt (1/76th of an inch)
  • Android: 1 dips (1px at 160dpi)
  • Microsoft: 1 DLG (Who knows?)

1 CSS px is 1/96in

Tap Targets - In the real world

1 CSS px is 1/96in

  • Suggested target size: (26 CSS pixels - 26/96 == 7mm)
  • Suggested margin: (8 CSS Pixels - 8/96 == 2mm)
